# Analisis Penyewaan Sepeda

Aplikasi ini dibangun menggunakan **Streamlit** untuk menganalisis data penyewaan sepeda berdasarkan berbagai faktor seperti hari libur, kondisi cuaca, dan hubungan antar variabel. Program ini juga menggunakan visualisasi data untuk menyampaikan hasil analisis dengan lebih baik.

## Cara Menjalankan Aplikasi Streamlit

### 1. Install Python dan Pip
Pastikan Anda telah menginstall **Python** dan **pip** (package manager untuk Python). Jika belum, Anda bisa mendownload dan menginstall Python dari [sini](https://www.python.org/downloads/).

Untuk memastikan Python dan pip sudah terinstall, jalankan perintah berikut di terminal atau command prompt:
```bash
python --version
pip --version

### 2. Install Streamlit
Install Streamlit dengan menjalankan perintah berikut di terminal atau command prompt:

```bash
pip install streamlit
```
### 3. Clone atau Download Repository
Clone repository ini ke komputer lokal Anda dengan perintah:

```bash
git clone https://github.com/staryzcode/dicoding-task.git
```
Atau Anda juga bisa download ZIP dari halaman GitHub dan ekstrak ke folder lokal.

### 4. Navigasi ke Direktori Proyek
Setelah repository terdownload atau ter-clone, buka terminal dan navigasi ke direktori proyek:

```bash
cd dicoding-task
```

### 5. Jalankan Aplikasi Streamlit
Jalankan aplikasi dengan perintah berikut di terminal:

```bash
streamlit run app.py
```
